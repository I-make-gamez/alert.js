Alert.js

A javascript framework developed by me, for me. I don't really care if you use it tho ;)


Get started:

Paste this link into the content of your head tag

<script src="https://cdn.imakegamez.com/alert.js"><script>

How To Use:

First, call the alert() function inside your code. an example of a prompt would be alert('prompt', 'Hello World', 'Hello World')

The prompt parameter saves the input into the ans variable. So make sure you don't have a variable saved called ans.

You can also call alert('alert', 'Hello World', 'Hello World') To show the same text w/o imput.

Errors:

Alert.js (Code: 1) : alert not recognized.: This means you have a syntax error in your alert you probabally misspelled alert or prompt.
