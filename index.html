Alert.js

A JavaScript library developed by me, for me. I don't really care if you use it tho ;)


Get started:

Paste this link into the content of your head tag

<script src="https://cdn.imakegamez.com/alert.js"><script>

How To Use:

First, call the alertJs() function inside your code. An example of a prompt would be alertJs('prompt', 'Hello World', 'Hello World')

The prompt parameter saves the input into a variable called ans. So make sure you don't have a variable called ans.

You can also call alertJs('alert', 'Hello World', 'Hello World') To show the same text w/o imput.

Errors:

Alert.js (Code: 1) : alert not recognized.: This means you have a syntax error in your alert. You probabally misspelled alert or prompt.
